## Editing these rules

This file is your global `~/CLAUDE.md`, generated by `~/src/personal/dotfiles/install.sh` from the section files in `~/src/personal/dotfiles/agent-skills/rules.d/`. When asked to remember a global rule or change your standing instructions, edit the matching section file there (or add a new one), re-run `./install.sh`, then commit the change in the dotfiles repo. Do not edit `~/CLAUDE.md` directly — the next install overwrites it. Machine-only rules go in `rules.d/99-local.md` (gitignored); a machine excludes a section by listing its slug in `SKIP_RULES` in `.dotfiles-local`.
